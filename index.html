Planifier son projet

  • Analyser les besoins
  • Distinguer les différents composants du projet
  • Estimer le temps de travail pour chaque partie
  • Créer le Modèle Conceptuel de Données (MCD) avec DBDesigner ou MySQLWorkbench ou autre
  • Organiser son code (fichiers et répertoires)
  • Définir les différentes tâches
  • Utiliser les outils nécessaires au travail en équipe (Git, partage de fichiers, etc.)
